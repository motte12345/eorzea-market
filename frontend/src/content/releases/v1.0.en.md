# v1.0 — Initial Release

**2026-03-25**

The first release of Eorzea Market.
Compare market board prices across all FFXIV servers in real-time and find cross-DC arbitrage opportunities.

## Features

### Watchlist
- Add items to your watchlist and view lowest prices by DC
- Displays lowest price and world name for each region (JP / NA / EU / OCE)
- Automatic arbitrage profit and rate calculation
- Customizable region selection and tax rate settings

### Item Search
- Incremental search with suggestions in Japanese and English
- Category-based item listings with price sorting

### Item Detail
- Full listing display with Region → DC → World hierarchy
- HQ / NQ filter
- Trade history
- Manual price refresh button (2-min cooldown)

### Rankings
- Most expensive items ranking
- JP ↔ NA price gap ranking (by rate / by amount)

### Data Collection
- Automatic periodic collection from Universalis API (Watchlist: 10 min / Others: 30 min)
- Covers all regions and DCs
