# v1.0 — 初回リリース

**2026-03-26**

Eorzea Market の初回リリースです。
FFXIV 全サーバーのマーケットボード価格をリアルタイムで比較し、DC間の転売差益を確認できます。

## 主な機能

### ウォッチリスト
- アイテムをウォッチリストに登録して、DC別の最安値を一覧表示
- JP / NA / EU / OCE のリージョン別に最安値とワールド名を表示
- リージョン間の差益額・利益率を自動計算
- 比較対象リージョンの選択、税率設定に対応

### アイテム検索
- 日本語・英語でのインクリメンタル検索（サジェスト付き）
- カテゴリ別のアイテム一覧・価格ソート

### アイテム詳細
- リージョン → DC → ワールドの階層構造で全出品を表示
- HQ / NQ フィルタ
- 売買履歴の表示
- 価格の手動更新ボタン（2分クールダウン）

### ランキング
- 高額アイテムランキング
- JP ↔ NA 価格差ランキング（利益率順 / 差額順）

### データ収集
- Universalis API からの定期自動収集（ウォッチリスト: 10分間隔 / その他: 30分間隔）
- 全リージョン・全DCのデータを網羅
